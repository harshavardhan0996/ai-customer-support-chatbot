# 🤖 AI Customer Support Chatbot

A smart chatbot built with OpenAI API, Python backend, and React frontend.

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Python
- AI: OpenAI GPT API

## Features
- Real-time AI responses
- Chat history
- Multi-language support

## Setup Instructions

### Backend
pip install -r requirements.txt
python app.py

### Frontend
cd frontend
npm install
npm start
